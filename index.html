About Myself

Ketertarikan saya pada teknologi dimulai ketika saya duduk di bangku SMA. Karena saya memiliki kesempatan untuk bermain video game untuk pertama kalinya, itu hanya berkembang saat saya mulai belajar cara menggunakan komputer dan menonton film fiksi ilmiah.
Ketika saya besar saya pergi ke Universitas untuk belajar ilmu teknik komputer, ketika saya mendapatkan laptop pertama saya dan akses reguler ke internet rasa ingin tahu saya semakin meningkat, selama tahun terakhir saya di Universitas saya bertanya pada diri sendiri apakah saya dapat menyebut diri saya seorang insinyur dan semacamnya jawaban yang menyakitkan adalah tidak.
Saya mulai mencari secara online cara belajar pemrograman dari situs web seperti SitePoint. Sayangnya, bahasa Inggris saya kurang bagus karena saya mencoba belajar pengembangan web dalam bahasa Inggris. Saya memulai kembali mempelajari pengembangan web, setelah mempelajari Bootstrap, setelah itu saya memutuskan untuk menghubungi perusahaan lokal untuk membangun situs web mereka dengan sedikit atau tanpa hasil.
Jadi pada tahun 2020 saya memutuskan untuk mendalami dasar-dasar seperti HTML, CSS, javascript dan mendapatkan pekerjaan jarak jauh untuk berkembang sebagai pengembang. percaya bahwa selama kita masih hidup, kita harus berusaha dan menjadi versi terbaik dari diri kita setiap hari!
